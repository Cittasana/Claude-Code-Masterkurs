<task type="auto">
  <name>Task 1: Configure SF Pro-style font stack and typography scale in Tailwind</name>
  <files>claude-code-masterkurs/tailwind.config.js</files>
  <action>
Extend the Tailwind theme with a comprehensive typography system:

**Font families:**
```js
fontFamily: {
  sans: ['-apple-system', 'BlinkMacSystemFont', 'SF Pro Display', 'SF Pro Text', 'Segoe UI', 'Roboto', 'Helvetica Neue', 'Arial', 'sans-serif'],
  mono: ['SF Mono', 'Menlo', 'Monaco', 'Consolas', 'Liberation Mono', 'Courier New', 'monospace'],
  display: ['SF Pro Display', '-apple-system', 'BlinkMacSystemFont', 'Segoe UI', 'Roboto', 'sans-serif'],
},
```

**Custom font sizes with Apple-style line heights:**
```js
fontSize: {
  // Display sizes - large headlines
  'display-xl': ['4rem', { lineHeight: '1.1', letterSpacing: '-0.02em', fontWeight: '700' }],    // 64px
  'display-lg': ['3rem', { lineHeight: '1.15', letterSpacing: '-0.02em', fontWeight: '700' }],   // 48px
  'display-md': ['2.25rem', { lineHeight: '1.2', letterSpacing: '-0.01em', fontWeight: '600' }], // 36px

  // Heading sizes
  'heading-lg': ['1.75rem', { lineHeight: '1.3', letterSpacing: '-0.01em', fontWeight: '600' }], // 28px
  'heading-md': ['1.375rem', { lineHeight: '1.35', letterSpacing: '-0.01em', fontWeight: '600' }], // 22px
  'heading-sm': ['1.125rem', { lineHeight: '1.4', fontWeight: '600' }], // 18px

  // Body sizes
  'body-lg': ['1.125rem', { lineHeight: '1.6' }],  // 18px - large body
  'body-md': ['1rem', { lineHeight: '1.6' }],      // 16px - default body
  'body-sm': ['0.875rem', { lineHeight: '1.5' }],  // 14px - small body

  // Caption/label sizes
  'caption': ['0.75rem', { lineHeight: '1.4', letterSpacing: '0.01em' }],  // 12px
  'overline': ['0.6875rem', { lineHeight: '1.3', letterSpacing: '0.08em', fontWeight: '600' }], // 11px
},
```

**Font weights:**
```js
fontWeight: {
  normal: '400',
  medium: '500',
  semibold: '600',
  bold: '700',
},
```

Keep all existing color configurations from Plan 01.

<task type="auto">
  <name>Task 2: Create typography utility classes for headlines, body, and captions</name>
  <files>claude-code-masterkurs/src/index.css</files>
  <action>
Add typography utility classes in `@layer components` section. Place these BEFORE the existing `.code-block` class:

```css
/* Typography utilities */
.text-display-hero {
  @apply text-display-xl font-display tracking-tight;
}

.text-display {
  @apply text-display-lg font-display tracking-tight;
}

.text-title {
  @apply text-display-md font-semibold tracking-tight;
}

.text-headline {
  @apply text-heading-lg font-semibold;
}

.text-subheadline {
  @apply text-heading-md font-semibold;
}

.text-body-large {
  @apply text-body-lg text-apple-text;
}

.text-body {
  @apply text-body-md text-apple-text;
}

.text-body-secondary {
  @apply text-body-md text-apple-textSecondary;
}

.text-caption {
  @apply text-caption text-apple-muted;
}

.text-overline {
  @apply text-overline uppercase text-apple-muted;
}

/* Prose-like content styling */
.prose-apple {
  @apply text-body-md text-apple-text leading-relaxed;
}

.prose-apple h1 {
  @apply text-display-lg font-bold mt-8 mb-4;
}

.prose-apple h2 {
  @apply text-heading-lg font-semibold mt-6 mb-3;
}

.prose-apple h3 {
  @apply text-heading-md font-semibold mt-4 mb-2;
}

.prose-apple p {
  @apply mb-4;
}

.prose-apple code {
  @apply bg-apple-surface px-1.5 py-0.5 rounded text-body-sm font-mono;
}
```

These utilities provide semantic typography classes that can be used throughout the app for consistent text styling.
