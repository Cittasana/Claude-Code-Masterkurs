# 🚀 Quick Start Guide

## 1️⃣ Projekt öffnen

```bash
cd claude-code-masterkurs
```

## 2️⃣ Dependencies installieren

```bash
npm install
```

Dies installiert alle benötigten Packages (~30 Sekunden).

## 3️⃣ Development Server starten

```bash
npm run dev
```

Der Server startet automatisch auf: **http://localhost:5173**

## 4️⃣ Browser öffnen

Öffne deinen Browser und navigiere zu:

```
http://localhost:5173
```

## ✅ Was du sehen solltest

1. **Dashboard** mit deinem Lernfortschritt
2. **Navigation** oben mit Dashboard, Lektionen, Projekte
3. **Streak Tracker** und Punkte-Anzeige
4. **Level Breakdown** (Level 1-3)
5. **Nächste Schritte** Sektion

## 🎓 Erste Schritte

### Lektion starten
1. Klicke auf "Nächste Lektion" im Dashboard
2. ODER navigiere zu **Lektionen** in der Navigation
3. Wähle **Lektion 0: Was ist Claude Code?**

### Quiz absolvieren
1. Scrolle zum Ende einer Lektion
2. Beantworte die Quiz-Fragen
3. Erhalte sofortiges Feedback
4. Sammle Punkte!

### Fortschritt verfolgen
- Dein Fortschritt wird automatisch gespeichert
- Kehre jederzeit zum Dashboard zurück
- Überprüfe deine Statistiken

## 🎨 Features ausprobieren

### Code kopieren
- Hover über Code-Blöcke
- Klicke auf das Copy-Icon
- Code wird in Zwischenablage kopiert

### Navigation
- Nutze Prev/Next Buttons zwischen Lektionen
- Markiere Lektionen als "Abgeschlossen"
- Springe direkt zu bestimmten Lektionen

### Quiz-System
- Multiple Choice Fragen
- Checklist Verifikation
- Hints bei Schwierigkeiten
- Max 3 Versuche pro Quiz

## 🔧 Troubleshooting

### Port already in use?
```bash
npm run dev -- --port 3000
```

### Module not found?
```bash
rm -rf node_modules package-lock.json
npm install
```

### Build failed?
```bash
npm run build -- --force
```
