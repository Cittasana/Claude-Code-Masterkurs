# Research Context — Claude Code Track

> Wöchentliche Research-Konfiguration für den **Claude Code** Track der
> Multi-Track-Masterkurs-Plattform.

## Track-Beschreibung

Claude Code ist Anthropics offizielle CLI für agentisches Programmieren mit
Claude. Der Track richtet sich an Entwickler, die Claude Code als
primäres KI-Coding-Tool im Terminal/in der IDE einsetzen.
