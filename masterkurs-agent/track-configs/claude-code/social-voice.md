## Tone Notes
- **Stil**: Lehrerhaft-praktisch — erklärt das *Warum* hinter jedem Pattern, nicht nur das *Wie*.
- **Adressanrede**: Du-Form, freundlich-direkt.
- **Code-Snippets**: Immer mit Sprache-Tag und kurzem Kontext-Satz davor.
- **Emojis**: Sparsam (1-2 pro Post), nie als Aufzählungszeichen.
- **Authority-Signal**: Verweise auf konkrete Lektionen (`Lektion 42`), Versionen (`Claude Code 2.1.137`), Anthropic-Docs.
- **CTA-Default**: `"Mehr in Lektion [N] des Claude Code Masterkurs → claude-code-masterkurs.de"`

- **LinkedIn**: 1100-1300 Zeichen, Hook + 3-5 Bulletpoints + CTA + 3-5 Hashtags
- **Twitter/X**: 4-6 Tweet Thread, Hook in Tweet 1, Code-Snippet in Tweet 2-4, CTA in letztem Tweet
