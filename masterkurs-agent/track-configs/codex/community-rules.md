## Welcome-Message

Hey {{username}}! Welcome to the **Codex Track** — OpenAI Codex CLI, multi-agent worktrees, and parallel-build workflows. Drop a quick intro and tell us what you're shipping with Codex right now.

## Allowed Topics

- Codex CLI (≥0.130) — install, config, multi-agent worktrees
- Codex MCP-Server, Tool-Use-Patterns, Sandbox-Modes
- OpenAI API + Codex-spezifische Modell-Auswahl (gpt-5-codex, o-series for code)
- Codex vs Claude Code vs Cursor vergleichende Diskussion (sachlich, kein Flame-War)
- Parallel-Build-Patterns, Tree-Editing, Diff-Review-Workflows

## Off-Topic (moderate — Vergleiche erlaubt, aber kein Anthropic-Bashing)

- Reine Claude-Code-Fragen ohne Codex-Bezug → `claude-code`-Track
- Lokale OSS-Modelle (außer Codex-OSS) → `local-llm`-Track
- Vergleichs-Threads sind erlaubt, müssen aber **respektvoll und faktenbasiert** sein
- Off-Topic-Schwelle: **moderate** (Codex + OpenAI-Tooling primär; Vergleiche willkommen wenn konstruktiv)

## Spam-Patterns (track-spezifisch flaggen)

- `gpt5-jailbreak`, `codex-cracked`, `openai-api-free`, `chatgpt-plus-free`
- Affiliate-Links auf "OpenAI Wrapper"-SaaS ohne Disclosure
- "Anthropic vs OpenAI"-Flame-Bait-Posts (low-effort, polemisch)
- Newly-joined accounts (<7d) mit Codex-Demo-Video von unbekanntem YouTube-Kanal
- "Codex-Killer"-Tools von obskuren Anbietern

## Engagement-Cadence

- **Monday 09:00 CET** — "Multi-Agent Monday" — Member shares a worktree-tree pattern or parallel-build setup
- **Wednesday 10:00 CET** — Codex Challenge (build X with Codex CLI, sandbox-mode constraint)
- **Friday 18:00 CET** — Diff-Review-Friday (post a Codex-generated diff, community reviews)

## Escalation Policy

- **Auto-Action**: Spam-Pattern-Match → delete + warn
- **Mod-Eskalation**: bei Vergleichs-Threads die in Flame-War kippen → Mod schließt Thread oder fordert Refactor
- **Founder-Eskalation**: bei systematischer Anti-Anthropic-Hetze (verstößt gegen Cross-Track-Respekt)
