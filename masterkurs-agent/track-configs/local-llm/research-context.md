# Research Context — Local LLM Track

> Wöchentliche Research-Konfiguration für den **Local LLM** Track der
> Multi-Track-Masterkurs-Plattform.

## Track-Beschreibung

Lokale LLMs (Ollama, LM Studio, llama.cpp, vLLM) ermöglichen privates,
offline-fähiges KI-Coding ohne Cloud-Abhängigkeit. Der Track richtet sich an
Entwickler mit Datenschutz-Anforderungen, Hardware-Enthusiasten und Hybrid-Setups
(lokal für Routine + Cloud für Heavy-Lifting).
